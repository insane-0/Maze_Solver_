========================
BOARD DESCRIPTION
========================
This PCB is designed for a compact autonomous maze-solving robot.
The board integrates:

- Microcontroller-based control system
- Dual DC motor driver interface
- Motor encoder input connectors
- Distance sensor connectors for wall detection
- Buck converter for regulated power supply
- Push buttons for user input
- Status LEDs for debugging and state indication

The PCB is optimized for robotics competitions and embedded system experiments.

========================
POWER DETAILS

- On-board buck converter generates logic voltage
- Motor supply and logic supply are routed separately
